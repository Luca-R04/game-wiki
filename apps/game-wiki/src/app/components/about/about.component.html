Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

WikiGames

WikiGames is een Single Page Application (SPA) waar informatie over iedereens favoriete games gedeeld wordt, iedereen kan zijn game naar keuze toevoegen om te laten lezen door anderen.

Entiteiten

WikiGames bevat 4 entiteiten: User, Game, Actor en Review.
Een user kan een alle 3 de andere entiteiten aanmaken, aanpassen en verwijderen als deze van de ingelogde user zijn.
Ook kan een user vrienden maken, dit wordt gedaan door een lijst met vrienden te embedden in de user.

Een game wordt gemaakt door een user, een game bevat reviews en actors en kan nog verwijderd worden als het deze al heeft.
Een game kan meerdere actors hebben en een actor kan ook in meerdere games zitten, een actor kan niet verwijderd worden als deze al in een game zit.
Als een user is ingelogd kan deze een review achterlaten, deze review wordt vervolgens bij een game gezet en wordt er voor de game een gemiddelde rating berekend.
Vervolgens kan een ingelogde user bij zijn profiel zien welke reviews deze user allemaal geschreven heeft.

User

  • id: number
  • naam: string
  • email: string
  • password: string
  • birthday: date

Game

  • id: number
  • naam: string
  • price: number
  • category: string
  • releaseDate: date
  • image: string
  • positivePercent: number

Actor

  • id: number
  • naam: string
  • birthDay: date
  • isMale: boolean

Review

  • id: number
  • reviewDate: date
  • isPositive: boolean
  • message: string
